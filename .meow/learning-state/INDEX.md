# Meow Ops Project Learning

This folder contains compact, owner-approved project knowledge. It is the canonical project-learning source for Codex, Claude Code, Hermes Agent, Google Antigravity, and Cursor.

## Read first

1. Read [constitution.md](constitution.md) for the authoritative project intention.
2. Read [current-state.md](current-state.md) for the active phase, priorities, and open decisions.
3. Load detailed decisions, practices, failures, or skills only when they are relevant to the current task.

## Governing rules

- Raw transcripts, private notes, secrets, and unapproved proposals must not be stored here.
- New learning is proposed from local evidence and becomes canonical only after owner approval.
- Project knowledge stays project-local unless the owner separately approves portfolio promotion.
- Native-agent files are adapters. This folder remains the canonical truth.
- Every published learning needs provenance, an expiry or review policy, and a rollback path.
